🎯 Hướng dẫn test

  1. Mở bất kỳ trang nào từ danh sách trên
  2. Tìm nút chatbot 💬 ở góc dưới bên phải (có hiệu ứng pulse)
  3. Nhấn vào nút chatbot để mở giao diện chat
  4. Test các câu hỏi sau:
    • "xin chào" - Test intent chào hỏi
    • "tôi muốn đặt hoa" - Test intent đặt hoa
    • "tư vấn cho tôi" - Test intent tư vấn
    • "cảm ơn bạn" - Test intent cảm ơn
    • "hỏi về đơn hàng" - Test intent tra cứu
    • "deal hot" - Test intent deal
  5. Kiểm tra console (F12) để xem log Dialogflow
  6. Test trên mobile để đảm bảo responsive
